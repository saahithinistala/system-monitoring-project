# Server Monitoring Project

This is a Python-based monitoring project developed for system health checks.

## Features

- Disk Usage Monitoring
- Service Status Check
- Log Error Detection
- API Health Check

## Project Structure

server_monitoring_project/
├── main.py
├── monitoring_script.py
├── application.log
└── logs/
    └── system_health_check_py.log

## Technologies Used

- Python
- requests
- subprocess
- datetime
- logging

## API Health Check

Checks:
- API reachability
- Status code validation
- Healthy / Unhealthy status
- Timeout handling
- DNS failure detection
- Connection issues
